These particles interact with each other, you can set the general system variables using the input panel. Resetting the number of particles or the number of classes resets the whole system. Also you can manage the interaction between all classes with the sliders, going from -1 up to 1, respectively rejecting or attracting other classes' particles. The first color of the slider is the class rejected from or attracted to the second color. The p value is the average momentum of the particles.
